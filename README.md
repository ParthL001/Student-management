# Student-management
🎓 Student Result Management System with Machine Learning
A Flask-based Student Result Management System integrated with Machine Learning to manage student records and predict marks using historical data stored in a MySQL database.
This project demonstrates the integration of Database Management Systems (DBMS) with Machine Learning in a web application.

📌 Features
Student Record Management
Add student records
View all student results
Search student by roll number
Edit student details
Delete student records
Machine Learning Integration
Predict student marks using historical database records
Uses Linear Regression algorithm
Trained automatically from stored data in MySQL
🧠 Machine Learning Concept
The project uses Linear Regression to predict student marks.
The model learns patterns from existing student records stored in the database and predicts marks for a given subject.

Input
Subject
Output
Predicted Marks
